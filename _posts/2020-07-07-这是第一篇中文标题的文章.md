### 一、前期准备
#### 1、安装Git
Git官方网站：https://git-scm.com
#### 2、安装Node.js
Node.js官方网站：http://nodejs.org
#### 3、安装Visual Studio Code
Visual Studio Code官方网站：https://code.visualstudio.com
#### 4、注册Github帐号
Github官方网站：http://github.com
#### 5、注册域名
(到域名运营商注册购买即可)

### 二、配置Git仓库
#### 1、创建仓库
打开Github官网，登陆之创建一个名为［用你的户名.github.io］的仓库
创建成功之后获得［用你的户名.github.io］二级域名，通过这个二级域名就可以直接访问站点
进入创建好的仓库，建立一个名为“CNAME”的文件，文件内容为你注册的域名

#### 2、设置SSH key
```
cd ~/. ssh   
ssh-keygen -t rsa -C "邮件地址"     
```
检查已存在的ssh密钥，如果提示 No such file or directory 是因为初次使用Git，确认三次会在用户目录生成 .ssh\id_rsa.pub 文件。

打开文件后，复制里面的长串字符。
登陆Github网站，点击头像－[Settings]－>[SSH and GPG keys]－>[New SSH key]
自定义Title，把文件里长串字符粘贴到Key的文本框内，点击［Add SSH key］

测试是否成功，在终端输入
```
ssh -T git@github.com
```
看到提示 Hi 你的用户名! You've successfully 表示成功
#### 3、设置Git远程帐号信息
在终端输入
```
git config --global user.name "换成Github的用户名"
git config --global user.email "换成Github的注册邮箱" 
```
### 三、Hexo博客程序
#### 1、安装Hexo
打开终端输入 
```
npm install -g hexo
```
安装过程结束后，在你理想的位置创建一个文件夹用来存放Hexo程序。

然后打开 Visual Studio Code 在左侧空档处右键选择“将文件夹添加到工作区”指向你刚刚创建存放Hexo的目录

然后对着刚刚添加的文件夹右键选择“在终端中打开”
依次输入以下命令
```
hexo init blog
cd blog
npm install
hexo server
```
提示访问：http://localhost:4000 
#### 2、配置Hexo
进入Blog目录下打开_config.yml文件，拉到最底修改以下内容。
```
deploy:
  type: git
  repository: 仓库地址
  branch: master
```
找以下内容到并修改
```
# URL
url: http://用户名.github.io/
root: /
```
全部配置好以后输入命令
```
npm install hexo-deployer-git --save    ＃安装部署插件
hexo g -d       ＃生成静态博客及部署到Github上。
```
至此可访问你的二级域名：用户名.github.io

### 四、域名绑定
虽域名运营商的后台设置多少有出入，但都大同小异。
进入域名后台管理，找到解析设置，添加两条记录：
主机记录 @和www 记录值填写 用户名.github.io
等待解析成功，注册域名即可访问。
