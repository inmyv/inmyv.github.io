# hugo-theme-xxx

## 简介
此为极简 Hugo 主题，无任何样式，您只需关注样式就可以完成 Hugo 主题的开发，也可以快速入门一些相关语法，是主题开发者的一个指明灯。

## 功能
1、极简代码；

2、支持最新版 Hugo；

3、快速入门；

4、给想开发主题人员提供思路；

## 教程
```
git clone https://github.com/UangTsui/hugo-theme-xxx.git themes/xxx
```

### 注意
1、在主题的根目录下有个 config.toml 文件，将这个 config.toml 文件复制到站点根目录，您可以根据自己的需求做相应的修改。

2、此主题使用的是 post，如：

```
hugo new post/1.md
```
